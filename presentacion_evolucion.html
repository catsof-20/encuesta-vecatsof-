Desincronización de Archivos

Ingeniería

Universal Database Driver

Implementamos un 'wrapper' en server.js que detecta el entorno y usa PostgreSQL o SQLite automáticamente.

async function runQuery(sql, params) { if (process.env.DATABASE_URL) { return await pool.query(sql, params); // Postgres } else { return localDb.prepare(sql).run(params); // SQLite } }
