Vecatsof v2.1:
Evolución en
la Nube

Este informe detalla los 12 pasos técnicos realizados para transformar Vecatsof en una plataforma de encuestas anónima de alto rendimiento distribuida globalmente.

Tecnología BaseNode.js / Express
InfraestructuraSupabase Cloud (PostgreSQL)
EstéticaNeo-Glassmorphism
PASO 1: UX

El Acceso Anónimo

Se eliminó el login obligatorio para reducir la fricción inicial. El registro se transformó en un sistema de sugerencias voluntario.

DECISIÓN: Priorizar la participación sobre la recolección forzada de datos.
PASO 2: CLOUD DB

PostgreSQL en Producción

Migración de SQLite local a una base de datos regional distribuida en Supabase para permitir acceso concurrente global.

MétricaSQLite (Local)Supabase (Cloud)
UbicaciónFísica (PC Local)Nube (AWS Paris)
Accesibilidad0% (Solo Local)100% (URL Pública)
PASO 3: SEGURIDAD

Fallo de Autenticación

Resolución del error de conexión causado por caracteres especiales en la contraseña de Supabase mediante codificación URI.

// CORRECCIÓN CLAVE: const password = encodeURIComponent('Cati&sofia20'); // Result: Cati%26sofia20
PASO 4: SERVIDOR

Gestión de Estáticos

Redirección de la ruta de archivos estáticos para evitar que el servidor sirviera versiones obsoletas desde carpetas de build antiguas.

app.use(express.static(path.join(__dirname)));
PASO 5: MIGRACIÓN

Esquema JSON Dinámico

Implementación de columnas tipo JSON en la base de datos para manejar opciones de respuesta y votos de forma escalable.

{ "options": ["Opción A", "Opción B", "Opción C"], "votes": [0, 5, 2] }
PASO 6: ARQUITECTURA

Driver Híbrido

Desarrollo de un ejecutor de consultas universal que permite al proyecto funcionar en local o en nube automáticamente.

async function runQuery(sql, params) { return process.env.DATABASE_URL ? await pool.query(sql, params) : localDb.prepare(sql).run(params); }
PASO 7: DINAMISMO

Escalado de Preguntas

Expansión del catálogo a 16 preguntas técnicas y activación de un selector aleatorio en el backend.

SQL: ORDER BY RANDOM() LIMIT 6
PASO 8: ALGORITMO

Cálculo de Perfil

Implementación de lógica en JS para procesar las respuestas y asignar uno de los 3 perfiles de personalidad dominantes.

const counts = [0, 0, 0]; userAnswers.forEach(ans => counts[ans]++);
PASO 9: UI MODERNA

Resultados Visuales

Integración de fondos fotográficos atmosféricos y efectos de transición para una experiencia de usuario gratificante.

PASO 10: ANALYTICS

Sincronización Global

Visualización de estadísticas globales en tiempo real, permitiendo al usuario comparar su perfil con el resto del mundo.

ESTADO FINAL

Conclusión Técnica

Vecatsof v2.1 ha alcanzado el estado de producción, con un 100% de éxito en la migración a la nube y arquitectura de datos optimizada.
